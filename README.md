# DSA Learning Dashboard

A personal DSA coach dashboard for coding interview prep.

## Run locally

```bash
npm install
npm run dev
```

Open the shown localhost URL in browser.

## Build

```bash
npm run build
npm run preview
```

## Customize

Edit `src/data/dsaData.js`:
- Add new problems in each topic `problems` array.
- Update 90-day roadmap in `roadmapPattern` or `roadmap90`.
- Add/edit topic videos in each topic `videos` array using YouTube embed URLs.
